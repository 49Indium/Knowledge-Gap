Welcome to Knowledge Gap!

Kowledge Gap is a tool to classify your notes automatically, find new connections between existing ideas and see a mind-map of everything you've learnt. It was made as part of the 2024 UQCS Hackathon, and what you're seeing is a live demo with Isaac Beh's notes. You can find more about Knowledge Gap here.

Some things to try:

  • Hover over nodes to see the concepts they represent
  • See automatically generated connections between concepts via the edges of the graph
  • Move nodes around by dragging
  • Move the graph around by dragging with the middle mouse button
  • Create new edges or delete exising ones by clicking between nodes
  • Change the forces (bottom left) and see how the graph responds
